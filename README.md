# Prism Next.js

This is the clean Next.js version of Prism.

## Correct GitHub structure

Your repository homepage must show:

```text
app/
package.json
tsconfig.json
next-env.d.ts
README.md
```

When you click `app`, it must show:

```text
globals.css
layout.tsx
page.tsx
```

## Upload through GitHub

1. Unzip this download.
2. Open the `prism-nextjs` folder.
3. Drag the entire `app` folder into GitHub.
4. Also drag these root files:
   - `package.json`
   - `tsconfig.json`
   - `next-env.d.ts`
   - `README.md`
5. Commit the changes.

Do not upload the outer `prism-nextjs` folder itself.
